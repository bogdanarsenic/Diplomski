<h2> How to start a project?  </h2>

<h6> Server Side: </h6>
<ol>
<li> Open "DiplomskiBack" in VisualStudio </li>
<li> Enter "update-database" in terminal to setup EntityFramework database </li>
<li> Run Ctrl+F5
</ol>

<h6> Angular Side: </h6>
<ol>
<li> Open "DiplomskiFront" </li>
<li> Enter "ng serve" in terminal </li>
</ol>
